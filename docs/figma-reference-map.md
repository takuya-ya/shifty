# Figma 参照マップ

**使いどき：** レイアウト・UIデザインをFigmaに合わせて実装・確認するとき。  
ロジックやAPI実装のみのタスクでは参照不要。  
該当タスクの行だけを読んで参照ファイルを特定すること。

ファイルはすべて `figma-file/app/components/` 以下にある。

---

## 共通

| コンポーネント | ファイル | 利用画面 |
|---|---|---|
| Sidebar（コンパクトサイドバー） | `Sidebar.tsx` | 管理者全ページ（compact=true で使用） |
| LogoutModal | `LogoutModal.tsx` | 全ページのログアウト処理 |

**Sidebar の注意点：**  
Figma では `compact={true}` (w-16) でアイコンのみ表示。ラベルなしのナビゲーション形式。  
画面仕様書の「ヘッダー → ナビゲーション」部分は、Figma では Sidebar に統合されている。
